# Oliveira Assessoria Jurídica

Repositório central para o projeto da **Oliveira Assessoria Jurídica**.

## Objetivo
Este projeto consiste em uma landing page completa e funcional para a Oliveira Assessoria, com foco em conversão e automação de leads.

## Estrutura do Repositório
- `/landing`: Contém os arquivos da interface da landing page (HTML, CSS, JS).
- `/agent`: Espaço reservado para integrações de agentes de IA.
- `/db`: Contém o schema do banco de dados (`schema.sql`).
- `/contracts`: Espaço reservado para modelos de contratos e documentos legais.

## Tecnologias Utilizadas
- HTML5 / CSS3 (Design Responsivo e Mobile-First)
- JavaScript Vanilla (Interatividade e Integração com Webhook)
- Fontes: Playfair Display e Source Sans 3 (via Google Fonts)
- Ícones: Font Awesome

## Como Executar Localmente
1. Clone este repositório.
2. Abra o arquivo `/landing/index.html` em seu navegador.
3. Para testar o formulário, certifique-se de que o webhook de destino está ativo.

## Detalhes Técnicos
- **Identidade Visual:** Paleta em Azul Marinho (#0A1628) e Dourado (#C9A84C).
- **Integração:** O formulário de contato envia os dados via POST para um webhook do n8n.
- **Responsividade:** Suporte completo para dispositivos móveis com botão flutuante de WhatsApp.
